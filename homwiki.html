command wiki

category command aliases description
utility /lightlevels /ll reveals which blocks mobs can spawn on
messaging /msg <username> <message> /m /pm /t /tell /w /whisper sends a private message to another player
messaging /msgtoggle /m /pm /t /tell /w /whisper blocks receiving all private messages
messaging /r /reply quickly reply to the last player to message you
utility /afk /away marks you as away-from-keyboard
teleport /back /return teleports you to your location prior to tp/spawn/warp
economy /balance /bal /money states the current balance of a player
claims /basicclaims /bc switches the shovel tool back to basic claims mode
miscellaneous /beezooka /beecannon throw an exploding bee at your opponent
utility /sethome /createhome Set home to your current location.
utility /sethome <name> /createhome <name> Set a home with a specific name. Necessary for setting multiple homes.
